# Product Overview

## Time-Based Progress Bars Application

A Next.js web application for tracking progress through both manual updates and automatic time-based progression. Users can create and manage different types of progress bars:

- **Manual Progress Bars**: Traditional progress tracking with user-controlled updates
- **Time-Based Progress Bars**: Automatic progression based on time passage, supporting:
  - Count-up bars from historical start dates to future targets
  - Count-down bars showing time remaining until deadlines
  - Arrival date bars tracking progress toward specific dates

## Key Features

- Real-time automatic updates for time-based progress
- Support for large time scales (up to 50 years)
- Historical start date support (up to 10 years in the past)
- Detailed progress statistics and completion tracking
- Unified interface for both manual and time-based progress tracking
- Responsive design with dark mode support

## Target Users

Individuals tracking long-term goals, project deadlines, life milestones, and any time-sensitive objectives that benefit from visual progress representation.
